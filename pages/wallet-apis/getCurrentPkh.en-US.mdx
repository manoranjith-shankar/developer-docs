# getCurrentPkh

Get the current [PKH DID](https://developers.ceramic.network/docs/advanced/standards/accounts/pkh-did/).

## Usage

```js
await dataverseConnector.wallet.getCurrentPkh();
```

## Parameters

void

## Returns

`string` - The current pkh:did

```js
did:pkh:eip155:80001:0xb4D93398f6F3FB5EE4436D1aE93b32d65693a799
```
