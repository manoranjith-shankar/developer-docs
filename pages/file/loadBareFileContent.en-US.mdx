# loadBareFileContent

Load the content of a bare file.

## Usage

```javascript
await dataverseConnector.runOS({
  method: SYSTEM_CALL.loadBareFileContent,
  params: indexFileId,
});
```

## Parameters

- `indexFileId`: `string` - the file id

## Returns

`Promise<string>` - the content of the file

```
data:image/png;base64,...
```
