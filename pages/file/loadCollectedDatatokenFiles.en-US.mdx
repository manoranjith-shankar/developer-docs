# loadCollectedDatatokenFiles

Get a list of files which has been collected as a data token.

## Usage

```js
await dataverseConnector.runOS({
  method: SYSTEM_CALL.loadCollectedDatatokenFiles,
});
```

## Parameters

void

<Callout>
  This function automatically gets the current DID from the wallet and returns
  the files which has been collected as a data token under this DID.
</Callout>
