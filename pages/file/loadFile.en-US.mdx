# loadFile

Load a file by its id.

## Usage

```javascript
await dataverseConnector.runOS({
  method: SYSTEM_CALL.loadFile,
  params: indexFileId,
});
```

<Callout type="info">
  Note that this is just an example. Executing this directly will fail because
  that fileId does not exist on your app. You need to replace the fileId with a
  file created by your app.
</Callout>

## Parameters

- `indexFileId`: `string` - the file id
