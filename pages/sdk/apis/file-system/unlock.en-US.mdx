# unlock

Unlock a monetized file/folder, first will check if the decryptionConditions have been meet,
if so, will mint a Collect NFT and unlock the file/folder.

## Usage
```js
await runtimeConnector.unlock({
  app,
  indexFileId:
    "kjzl6kcym7w8y5xf0j7kmies5i30f86xlerbfp42qnxju2vfno1io6i04sy2p2d",
});

```

## Parameters
```js
{
    app?: string;
    streamId?: string;
    indexFileId?: string;
}
```
- `app`: `string` - The app name
- `streamId`: `string` - The stream id if you are unlocking a data stream
- `indexFileId`: `string` - The index file id if you are unlocking a file

## Returns
```js
CollectOutput &
Partial<{
    newFile: MirrorFile;
    isCurated: boolean;
    currentFolder: StructuredFolder;
    allFolders: StructuredFolders;
}>

interface CollectOutput {
  datatokenId: string;
  collector: string;
  collectNFT: string;
  tokenId: BigNumberish;
  txHash: string;
}
```

    This function will send a transaction to mint a NFT, currently on Mumbai(Polygon Testnet).
