# updateFolderBaseInfo

update the metadata of a folder


## Usage
```js
await runtimeConnector.updateFolderBaseInfo({
    folderId,
    newFolderName,
    newFolderDescription,
});
```

## Parameters
```js
{
    folderId: string;
    newFolderName?: string;
    newFolderDescription?: string;
    syncImmediately?: boolean;
}
```
- `folderId`: `string` - id of the folder to update
- `newFolderName`: `string` - new name of the folder
- `newFolderDescription`: `string` - new description of the folder
- `syncImmediately`: `boolean` - whether to sync the update immediately or wait for os scheduled sync

## Returns
```json
{
    currentFolder: StructuredFolder;
    allFolders: StructuredFolders;
}
```
