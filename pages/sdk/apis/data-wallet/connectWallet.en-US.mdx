# connectWallet

Connect a wallet.

## Usage
```js
import { WALLET } from "@dataverse/dataverse-connector";

const pkh = await dataverseConnector.connectWallet(WALLET.METAMASK);
```

## Parameters
`enum WALLET`
```js
enum WALLET {
  METAMASK = "MetaMask",
  WALLETCONNECT = "WalletConnect",
  COINBASE = "Coinbase",
  PARTICLE = "Particle"
}
```

## Returns
```json
{
    "address": "0x312eA852726E3A9f633A0377c0ea882086d66666",
    "chain": {
        "chainId": 80001,
        "chainName": "mumbai"
    },
    "wallet": "MetaMask"

```
