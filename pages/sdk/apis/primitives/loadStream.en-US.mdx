# loadStream

load a data stream by stream id

## Usage
```javascript
await runtimeConnector.loadStream({
  app: "app-name",
  streamId: "kjzl6kcym7w8ya8g9cj5764kll6veldqws40ub29r1odut2pxiqxet80sl9p6th",
});
```

## Parameters
- `app`: `string` - the name of the app
- `streamId`: `string` - the stream id
