# loadStream

Load a data stream by streamId

## Usage

```javascript
await dataverseConnector.loadStream({
  streamId: 'kjzl6kcym7w8ya8g9cj5764kll6veldqws40ub29r1odut2pxiqxet80sl9p6th',
});
```

<Callout type="info">
  Note that this is just an example. Executing this directly will fail because
  that streamId does not exist on your app. You need to replace the streamId
  with a data stream created by your app.
</Callout>

## Parameters

- `streamId`: `string` - the stream id
