# createCapability

Function for asking for user's authorization to user's data.

## Usage

```js
import { RESOURCE } from '@dataverse/dataverse-connector'

await dataverseConnector.createCapability({
  app,
  resource: RESOURCE.CERAMIC,
})
```

## Parameters
```js
{
  app: string;
  resource?: RESOURCE;
}
```
- `app`: `string` - which app is asking for authorization.
- `resource`: `RESOURCE` - Resource to give access to the capability.
  ```js
  enum RESOURCE {
    CERAMIC,
  }
  ```
