# loadDatatokens

Get the detailed info of a list of data tokens by their ids(Data Token id is the contract address of the data token).

## Usage

```js
await dataverseConnector.runOS({
  method: SYSTEM_CALL.loadDatatokens,
  params: [
    '0x6293BA7114867Ee79751d36376B969168DFD01d2',
    '0xcD743313EA468bb7FCf075cfe7cBcB3880C2674E',
  ],
});
```

## Parameters

- `datatokenIds`: `Array<string>` - a list of data token ids.
