# loadDatatokensCreatedBy

Get the list of data tokens by creator address.

## Usage

```js
await dataverseConnector.runOS({
  method: SYSTEM_CALL.loadDatatokensCreatedBy,
  params: address,
});
```

## Parameters

- `address`: `string` - wallet address of data token creator.

## Returns

List of data tokens.
