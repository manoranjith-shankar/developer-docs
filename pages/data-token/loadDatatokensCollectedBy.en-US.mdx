# loadDatatokensCollectedBy

Get the list of data tokens collected by a wallet address.

## Usage

```js
await dataverseConnector.runOS({
  method: SYSTEM_CALL.loadDatatokensCollectedBy,
  params: address,
});
```

## Parameters

- `address`: `string` - wallet address of data token collector.

## Returns

List of data tokens.
