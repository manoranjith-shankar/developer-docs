# loadDatatoken

Get the detailed info of a data token by its id(Data Token id is the contract address of the data token).

## Usage

```js
await dataverseConnector.runOS({
  method: SYSTEM_CALL.loadDatatoken,
  params: datatokenId,
});
```

## Parameters

- `datatokenId`: `string` - data token id.

## Returns

Data token info.
