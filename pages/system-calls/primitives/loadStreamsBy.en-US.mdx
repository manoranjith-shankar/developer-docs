# loadStreamsBy

Load multiple streams under a data model

## Usage
```js
await runtimeConnector.loadStreamsBy({
  modelId,
  pkh
});
```

## Parameters
```js
{
  modelId: string;
  pkh?: string;
}
```
- `modelId`: `string` - the data model id
- `pkh?`: `string` - user did, if not provided, will return all streams under the data model
