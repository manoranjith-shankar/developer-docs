# loadFolderTrees

Retrieve all folders under an app

## Usage

```js
await dataverseConnector.runOS({
  method: SYSTEM_CALL.loadFolderTrees,
});
```

## Parameters

void

## Returns

```
StructuredFolderRecord
```
