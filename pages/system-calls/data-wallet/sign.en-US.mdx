# sign

To open a popup and sign a message.

## Usage

```js
import { SignMethod } from '@dataverse/dataverse-connector';

await dataverseConnector.sign({
  method: SignMethod.signMessage,
  params: ['test'],
});
```

## Parameters

```js
{
    method: string;
    params: any[];
}
```

- `method`: `string` - specifies which `sign` method to call. such as `signMessage`.
- `params`: `any[]` - specifies message content to sign.

A hash of the signed message.
