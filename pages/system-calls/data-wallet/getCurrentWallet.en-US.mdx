# getCurrentWallet

Get the current wallet connected.

## Usage

```javascript
await dataverseConnector.getCurrentWallet();
```

## Parameters

void

## Returns

```js
Promise<{
    address: string;
    chain: Chain;
    wallet: WALLET;
}>
```

```json
{
  "wallet": { "name": "MetaMask", "type": "CRYPTO_WALLET" },
  "address": "0xb4D93398f6F3FB5EE4436D1aE93b32d65693a799",
  "chain": { "chainId": 137, "chainName": "polygon" }
}
```
