# isCollected

Get whether a Datatoken has been collected by a wallet address.

## Usage

```js
await dataverseConnector.isCollected({
  datatokenId,
  address,
});
```

## Parameters

- `string` - data token ID.
- `string` - wallet address.

## Returns

```js
Promise<boolean>
```

```json
true
```
