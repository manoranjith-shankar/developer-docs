# contractCall

You can call a contract function with this method via our SDK.

## Usage

```js
await dataverseConnector.contractCall({
  contractAddress: '0xB07E79bB859ad18a8CbE6E111f4ad0Cca2FD3Da8',
  abi: [
    {
      inputs: [],
      name: 'metadata',
      outputs: [
        {
          components: [
            {
              internalType: 'address',
              name: 'hub',
              type: 'address',
            },
            {
              internalType: 'uint256',
              name: 'profileId',
              type: 'uint256',
            },
            {
              internalType: 'uint256',
              name: 'pubId',
              type: 'uint256',
            },
            {
              internalType: 'address',
              name: 'collectModule',
              type: 'address',
            },
          ],
          internalType: 'struct IDataToken.Metadata',
          name: '',
          type: 'tuple',
        },
      ],
      stateMutability: 'view',
      type: 'function',
    },
  ],
  method: 'metadata',
  params: [],
});
```

## Parameters

```js
{
    contractAddress: string;
    abi: any[];
    method: string;
    params: any[];
    mode?: Mode;
}

enum Mode {
  Read,
  Write,
}
```

- `contractAddress`: `string` - The address of the smart contract.
- `abi`: `any[]` - The abi of the smart contract.
- `method`: `string` - The method name you want to call.
- `params`: `any[]` - The parameters of the method.
- `mode`: `Mode` - The mode of the call. Default is `Mode.Read`. Do remember that if you want to write to the blockchain, you need to use `Mode.Write`.

<Callout type="info" emoji="ℹ️">

  <a href="/sdk/apis/data-wallet/connectWallet">connectWallet</a>
</Callout>
